## 🚀 How to Use

1. Clone the repository
2. Open `index.html` in a web browser
3. All social links are fully functional and open in new tabs
4. Music will autoplay and loop on page load

## 🎨 Customization

### Change Music Video
In `index.html`, find the music section and replace the YouTube embed ID:
```html
<iframe src="https://www.youtube.com/embed/YOUR_VIDEO_ID?autoplay=1&loop=1&playlist=YOUR_VIDEO_ID"></iframe>
```

### Adjust Colors
In `styles.css`, modify the CSS variables:
```css
:root {
    --primary-color: #dc143c;  /* Red */
    --secondary-color: #1a1a1a; /* Black */
    --accent-color: #ff4444;
}
```

### Add More Social Links
Add buttons in the social section with appropriate icons and links.
